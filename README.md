👤 Profile Card UI

✨ Descrição do projeto:

Este projeto consiste em uma interface de cartão de perfil (Profile Card UI) desenvolvida com HTML e CSS. O objetivo é apresentar informações básicas de um usuário de forma visual, moderna e organizada.

O cartão inclui imagem de perfil, nome, profissão, ícones de redes sociais, botões de interação e indicadores de engajamento como curtidas, comentários e compartilhamentos.

A interface utiliza um design simples e responsivo, com foco em layout centralizado, tipografia moderna e elementos visuais limpos.

---

🎯 Objetivo
- Praticar a estruturação de páginas web utilizando HTML5.
- Aplicar estilos visuais e layout utilizando CSS3.
- Desenvolver um componente de interface reutilizável.
- Aprender organização de arquivos em projetos front-end.
- Criar um layout moderno de **cartão de perfil para aplicações web**.

---

🧠 Tecnologias utilizadas

- HTML5 – Estrutura da página  
- CSS3 – Estilização e layout  
- Google Fonts (Poppins) – Tipografia do projeto  
- Boxicons – Ícones para redes sociais e indicadores
